
## Prerequisites

* Node 8
* Git

## Setup

Clone the repo and install the dependencies.

```bash
git clone https://github.com/ishandeveloper/MERN-Stack-Development.git
cd Notes\{Folder Name}\

npm install
```
To run the app, simply write

```bash
node index.js
```
